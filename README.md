# BlackValley

A Stardew Valley battle mod prototype based on:
- deck-based combat
- protecting farmland
- monster infection mechanics

## Current Prototype Scope
- open battle menu with hotkey
- overlay battle UI
- end turn button
- 4x4 field area
- farmer + slime + 3 card slots

## Run
1. build the mod
2. launch Stardew Valley through SMAPI
3. press the configured hotkey to open battle menu
